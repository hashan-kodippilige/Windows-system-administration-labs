# Windows System Administration Labs

## Overview

This repository contains hands-on laboratory exercises completed as part of a Windows System Administration course.

The labs focus on Windows operating system deployment, virtualization, networking, security configuration, storage management, performance monitoring, and remote access technologies.

## Skills Demonstrated

- VMware Workstation
- Windows 10 Administration
- Virtual Machine Deployment
- User and Group Management
- Security Policy Configuration
- AppLocker
- IPv4 Networking
- Storage Spaces and VHD Management
- Registry Administration
- Performance Monitoring
- Remote Desktop and VPN Configuration
- PowerShell Fundamentals
